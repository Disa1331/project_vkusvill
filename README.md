Автоматизация расчета и визуализации ключевых метрик мобильного приложения

Построен пайплайн для расчета и визуализации ключевых метрик мобильного приложения:
• Активности пользователей
• Заказов
• Выручки

Стек технологий:
• Python (pandas, numpy, sqlite3, gspread)
• SQL (CTE, CASE, агрегаты)
• Airflow
• Google Sheets API
• Yandex DataLens

Что сделано:
• Очистка и подготовка данных
• Построение агрегированных витрин
• SQL-запросы с использованием CTE и CASE
• Автоматизация выгрузки данных в Google Sheets
• Настройка интерактивного дашборда
• Автоматизация обновления данных через DAG в Airflow

Результат:
Аватоматически обновляемый дашборд с ключевыми метриками и витрины для масштабирования аналитики.
