# Автоматизация расчета и визуализации ключевых метрик мобильного приложения

## Краткое описание
Построен пайплайн для расчета и визуализации ключевых метрик мобильного приложения:
- Активности пользователей
- Заказов
- Выручки

## Стек технологий:
- **Python**: pandas, numpy, sqlite3, gspread
- **SQL**: CTE(WITH), CASE, агрегатные функции
- **Airflow**
- **Google Sheets API**
- **Yandex DataLens**

## Данные
- Таблица `events` (действия пользователей в приложении)
- Таблица `orders` (данные о заказах и выручке)
- Период анализа: 1 месяц
- Основные поля: Дата, ID пользователя, Тип действия, ID продукта, ID заказа, Выручка

## Что сделано:
- Очистка и подготовка данных (проверка типов, удаление пробелов)
- Построение агрегированных витрин для расчета ключевых метрик
- SQL-запросы с использованием CTE(WITH), агрегатных функций и CASE
- Автоматизация выгрузки данных в Google Sheets через API
- Настройка интерактивного дашборда в Yandex DataLens
- Автоматизация обновления данных через DAG в Airflow

## Результат:
- Автоматически обновляемый дашборд с ключевыми метриками приложения
- Пайплайн в Airflow, обеспеивающий ежедневное обновление данных без ручного вмешательства
- Витрины данных, пригодные для дальнейшего мастабирования аналитики

## Выводы
- Построена автоматизированная система обработки и визуализации данных
- Обеспечена регулярная аналитика ключевых метрик без ручного вмешательства

## Рекомендации
- Использовать витрины данных как единый источник аналитики
- Расширять набор метрик и подключать новые источники данных
- Переодически пересматривать структуру витрин и дашборда по мере изменения бизнес-задач
